# 源码目录树骨架模板

当你要将你的源码以树的形式展示出来的时候，下面骨架就能用上了。

```js
project
├── dist
│   ├── css
│   │   └──  ...
│   │
│   └── js
│       ├── ...
│       │   ├── ...
│       │   ├── ...
│       │   └── ...
│       │
│       ├── ...
│       │   ├── ...
│       │   ├── ...
│       │   └── ...
│       │
│       ├── ...
│       └── ...
│
└── src
    ├── css
    │   ├── ...
    │   │   ├── ...
    │   │   ├── ...
    │   │   └── ...
    │   │
    │   ├── ...
    │   └── ...
    │
    └── js
        ├── ...
        │   ├── ...
        │   ├── ...
        │   ├── ...
        │   ├── ...
        │   └── ...
        │
        ├── ...
        │   ├── ...
        │   ├── ...
        │   └── ...
        │
        ├── ...
        │   ├── ...
        │   ├── ...
        │   ├── ...
        │   └── ...
        │
        ├── ...     // ...
        └── ...    // ...
```
